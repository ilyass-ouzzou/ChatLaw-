Introduction
-----------

Le Chatbot des Codes Marocains est une application web interactive permettant aux utilisateurs
de poser des questions sur les codes juridiques marocains et d'obtenir des réponses instantanées.

Fonctionnalités principales
-------------------------

* Recherche sémantique dans les codes juridiques
* Interface utilisateur intuitive
* Support multilingue (français et arabe)
* Mode sombre
* Export des conversations en PDF
